Panorama: Scalable Nearest Neighbor Search In High Dimensions UNDER REVIEW
Venue: ICML 2025 | Collaborators: Dr. Panagiotis Karras, University of Copenhagen

Designed Panorama, an exact k-nearest neighbor search algorithm that mitigates the curse of dimensionality via DCT energy compaction and Cauchy-Schwarz bounds. The algorithm achieves O(n log d) query complexity by progressively pruning distance computations in high-dimensional spaces (d ≈ 10⁶).

Key Contributions:
• Novel dimension pruning using DCT and Haar Wavelets
• 10× speedup over ANNOY, HNSW baselines on standard datasets
• Linear scalability with parallel architecture (k-fold speedup with k workers)
• Particularly effective for transient data in ML distillation pipelines

Technical Innovation:
• Energy compaction through Discrete Cosine Transform
• Cauchy-Schwarz inequality for bound computation
• Iterative zooming algorithm for progressive refinement
• Shared memory architecture for parallel processing

Experimental Results:
• Tested on datasets up to 10,000 dimensions
• Consistent performance improvements across different data types
• Memory-efficient implementation suitable for large-scale applications
• Superior performance on image classification and location-based services

Status: Manuscript submitted to ICML 2025 - under peer review

Completion: Dec 2023 | Collaborators: Dr. Preeti Panda

Algorithms Implemented:

Completion: Nov 2023 | Collaborators: Dhruv Joshi

Performance Results:
